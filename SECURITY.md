# Security

`rm-wiki` is a frontend-only AppLoad application. It talks to Wikipedia endpoints directly from the device and stores only local reader state such as history, cache entries, and typography preferences.

## Reporting

Please report security issues through GitHub private vulnerability reporting if it is available for this repository. If it is not available, open a minimal public issue that states there is a security concern without including exploit details.

## Supported Versions

Only the latest release is supported.
