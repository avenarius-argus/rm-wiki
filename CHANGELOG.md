# Changelog

## v0.1.0 - 2026-04-22

Initial public release.

- AppLoad frontend for Xovi-managed reMarkable Paper Pro and Paper Pro Move devices.
- Explicit Wikipedia search, article results, paged article reading, and recent history.
- Paper-first reader controls with persistent type size and minimal e-ink visual treatment.
- Host-side tests for API normalization, cache behavior, history limits, pagination, and AppLoad package contract.
- Release archive format for direct AppLoad installation.
- Vellum package recipe for package-manager submission.
